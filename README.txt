== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'ultra menu remove'
3. Activate ultra menu remove from your Plugins page.


= From WordPress.org =

1. Download ultra menu remove.
2. Upload the 'Ultra Menu Remove' directory to your 'wp-content/plugins/ultra-menu-remove.zip' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate ultra menu remove from your Plugins page.
